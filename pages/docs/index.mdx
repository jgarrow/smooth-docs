# Welcome

## Getting Started

Edit `docs/index.mdx` this page to start writing documentation.

## Live Code Editor

Just add `live` to the head of the code block to make it a live code editor! See the [docs](https://smooth-doc.com/docs/react-live-editor/) for more details.

```js live
<p style={{ color: 'rebeccapurple' }}>Change me!</p>
```
